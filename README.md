# tab-rotator

タブを一定間隔で順番に切り替えるツールです。

## 使い方

1. リポジトリを取得します。
2. アプリケーションを拡張機能としてブラウザに追加します。 追加の方法はブラウザのドキュメントを参照してください。
3. 拡張機能のアイコンをクリックして、設定画面を開きます。
4. 切り替え間隔を秒単位で設定します。
5. 「開始」ボタンをクリックして、自動切り替えを開始します。
6. 「停止」ボタンをクリックして、自動切り替えを停止します。
## 基本操作

- **開始**: 自動切り替えを開始します。
- **停止**: 自動切り替えを停止します。
- **設定変更**: 切り替え間隔や対象タブを変更します。
